##Инструкциия по примению
1. Сколонируйте репозиторий.
    ```
    $ git clone git@github.com:trpo2023/cw-ip_213_pwgen.git
    ```
2. Сборка проекта.
    ```
    $ make
    $ make run ARGS="[PW_LEN] [NUM_OF_PW] [OPTION]"
    ```
3. Удаление артефактов сборки.
    ```
    $ make clean
    ```
